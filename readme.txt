sham
~~~~

Like 'make' but without the Makefile.

   Usage: sham [command]
   e.g.   sham gcc test.c -o test.o

   sham will only execute the command if it's dependencies aren't up to date.

You need the Detours library (version 1.5) to compile.
http://research.microsoft.com/sn/detours/
